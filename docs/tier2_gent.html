End-of-life for victini Tier-2 cluster (12 Feb'24)

Start: 2024-01-23 10:00:00
End: 2024-02-23 20:00:00
Updated: 2024-01-28 10:05:18
Level: low

The HPC-UGent Tier-2 cluster victini will be shut down mid February 2024, to make room for a new cluster.

Only jobs that are guaranteed to finish by Mon 12 Feb'24 00:00 CET will still start.

If you are still relying on victini, please migrate your workloads to another cluster ASAP.

Possible instability of login nodes, debug nodes, webportals, Globus

Start: 2023-12-13 10:00:00
End: Unknown
Updated: 2024-01-28 10:05:18
Level: medium

We have uncovered an instability issue on the infrastructure supporting the login nodes, debug nodes, webportal and Globus. On unpredictable moments, this issue might cause a node or service to hang, forcing us to restart it. This issue does not affect your jobs. We are further investigating the issue in search for a fix.

But for the time being, please report to hpc@ugent.be when you notice a login node, debug node, webportal or Globus is unavailable to you.
