Security shutdown of access to Tier2 systems

Start: 2025-09-12 14:00:00
End: Unknown
Updated: 2025-09-15 17:39:49
Level: high

Access to the Brussels Tier2 systems is currently shut down. This is in response to a major security flaw in the kernel that was recently identified.

Access limitation applies to:

  • All VUB Tier2 login nodes
  • The VUB web portal portal.hpc.vub.be
All these systems will not be reachable. However, running jobs will not be affected.

All fixes are now ready for deployment. Cluster nodes will be upgraded as soon as they become idle. We expect login nodes to be back online by the end of Monday, 15th September, with the full Hydra cluster likely restored by Tuesday, 16th September. We can confirm that the security of Hydra has not been compromised.

For more details, please check our news page
