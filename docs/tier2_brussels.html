Security shutdown of access to Tier2 systems

Start: 2025-09-12 14:00:00
End: Unknown
Updated: 2025-09-14 20:20:40
Level: high

Access to the Brussels Tier2 systems is shut down with immediate effect. This is in response to a major security flaw in the kernel, which has been recently identified.

Access limitation applies to:

  • all VUB Tier2 login nodes
  • the VUB web portal portal.hpc.vub.be
All these systems will not be reachable. However, running jobs will not be affected.

In the next few days, we will perform emergency maintenance procedures. If all goes well, the systems should be back up in the beginning of next week.
